How to use this tool

To calculate damage during combat use the cards below to setup your Attacker and Blockers. A card's Power value goes in the left box and Toughness value in the right. Select any abilities that are in play. If multiple blockers are present, damage is dealt sequentially. When ready, click the Battle button!
